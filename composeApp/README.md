# Compose Multiplatform Preview

This module is a lightweight frontend scaffold for:

- Android simulator preview via `WebView`
- iOS simulator preview via `WKWebView`
- Shared Compose dashboard shell

It expects the backend to serve the provided marketing page at:

`http://localhost:8080/site/index.html`

Android emulator uses:

`http://10.0.2.2:8080/site/index.html`
